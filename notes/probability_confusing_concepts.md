# 🧠 Probability Theory: Confusing Concepts Cleared — Summary

이 정리본은 확률론을 공부하면서 가장 헷갈리기 쉬운 수학적 정의의 괴리, 모호한 표기법의 실체, 그리고 용어 간의 매핑 관계를 명확히 짚고 넘어 가기 위해 작성된 개념 FAQ 노트임.

---

## 1. 확률 변수(Random Variable)는 왜 '변수'가 아니라 '함수'인가?
* **초기의 직관(변수)**: 확률론 초기에는 "동전이나 주사위를 던졌을 때 어떤 값이 나올지 모르는, 무작위로 변하는 수"로 직관적으로 다룸. 그래서 **'무작위 변수(Random Variable)'**라는 명칭이 굳어짐.
* **현대 수학적 정의(함수)**: 수학적으로 엄밀한 계산과 확률 측정이 가능하도록 하기 위해 표본 공간 $\Omega$의 결과(Outcome, $\omega$)를 실수 $\mathbb{R}$로 변환해 주는 **함수**로 정의함.
    $$X: \Omega \rightarrow \mathbb{R}$$
* **예시**: 동전 던지기 $\Omega = \{\text{앞면}, \text{뒷면}\}$ 일 때,
    * $X(\text{앞면}) = 1$
    * $X(\text{뒷면}) = 0$
    * 수학적으로 $X$는 매핑 규칙(함수)이지만, 실제 수식에서는 $\omega$를 생략하고 마치 일반 대수학의 미지수/변수처럼 $X$ 하나로 표현하여 대수 연산($X + Y$, $X^2$)을 수행함.

---

## 2. 확률 함수 $P$는 함수 $X$를 입력으로 받는 합성함수인가?
* **질문의 요지**: 확률 변수 $X$도 함수이고, 확률 $P$도 함수인데, 우리는 $P(X \le x)$처럼 $P$의 입력 안에 $X$를 집어넣음. 그렇다면 함수가 함수를 대입받는 형태(합성함수 등)인 것인가?
* **수학적 사실**: **그렇지 않음.** 확률 측도 $P$의 입력(정의역)은 언제나 **'집합(사건)'**임.
* **표기법의 축약**: $P(X \le x)$는 수식의 가독성을 높이기 위해 복잡한 집합 표현을 축약한 대수적 약속임.
    $$P(X \le x) \quad \Longleftrightarrow \quad P\Big( \big\{ \omega \in \Omega \mid X(\omega) \le x \big\} \Big)$$
* **작동 메커니즘**:
    1. 함수 $X$를 통해 특정 조건($X(\omega) \le x$)을 만족하는 표본들을 모아 **집합(사건 $A$)**을 정의함.
    2. 생성된 **집합 $A$**를 함수 $P$에 입력함.
    3. 함수 $P$가 그 집합의 가능성(측도)을 나타내는 **실수 확률값**을 출력함.
* **결론**: $P$가 함수 $X$ 자체를 직접 입력받는 것이 아니며, 단지 함수 $X$를 통해 선별된 **'집합(사건)'**을 입력받을 뿐임.

## 3. 집합론 용어 vs 확률론 용어 1:1 매핑
확률론은 측도론과 집합론 위에 세워졌기 때문에, 집합론의 용어들이 확률론에서는 고유한 의미를 가진 새로운 이름으로 불림. 이를 혼동하지 않는 것이 중요함.

| 집합론 (Set Theory) | 확률론 (Probability) | 기호 / 예시 | 의미 |
| :--- | :--- | :--- | :--- |
| **전체집합 (Universal Set)** | **표본 공간 (Sample Space)** | $\Omega$ or $S$ | 실험에서 일어날 수 있는 모든 결과의 집합 |
| **원소 (Element)** | **표본 / 결과 (Outcome)** | $\omega$ | 표본 공간에 속한 개별적인 결과 |
| **부분집합 (Subset)** | **사건 (Event)** | $A, B, E$ | 우리가 관심을 가지는 결과들의 모임 (확률을 계산하는 대상) |
| **서로소 집합 (Disjoint Sets)** | **배반 사건 (Mutually Exclusive)** | $A \cap B = \emptyset$ | 동시에 일어날 수 없는 두 사건 |
| **원소의 개수 (Cardinality)** | **경우의 수** | $|A|$ | 사건에 속한 원소의 수 (등확률 공간 확률 계산의 기초) |

## 4. 확률변수 $X$에 대해 $P(X)$라고 쓰는 것은 왜 틀린 표현인가?
* **질문의 요지**: 확률변수를 $X$라고 할 때, $P(X)$는 왜 올바르지 않은 수학적 표기인가?
* **이유 (입력 타입의 불일치)**:
    * 함수에 값을 대입할 때는 입력 대상의 수학적 성격(타입)이 일치해야 함.
    * 확률 함수 $P$의 입력 타입은 **'집합(사건)'**임.
    * 확률 변수 $X$의 수학적 실체는 **'함수($X: \Omega \rightarrow \mathbb{R}$)'**임.
    * 따라서 $P(X)$라고 적는 것은 집합을 받아야 하는 함수 $P$에 또 다른 함수 $X$ 자체를 직접 대입한 꼴이 되므로 타입 오류임.
* **올바른 표현 방법**:
    * 괄호 안에는 반드시 **집합(사건)**을 의미하는 조건문 형태가 와야 함.
    * 예시:
        * $P(X = x)$ : $X(\omega) = x$를 만족하는 표본들의 **집합**에 대한 확률 (O)
        * $P(X \le x)$ : $X(\omega) \le x$를 만족하는 표본들의 **집합**에 대한 확률 (O)
        * $P(X \in A)$ : $X(\omega)$가 실수의 부분집합 $A$에 속하도록 하는 표본들의 **집합**에 대한 확률 (O)

## 5. $P(X = x_k)$와 $P_X(x_k)$는 결과가 같은데 무엇이 다른가?
* **질문의 요지**: 두 기호는 수학적으로 완전히 동일한 값을 출력하지만, 어떤 개념적 관점의 차이가 있는가?
* **핵심 차이 (개념적 층위의 차이)**:
    1. **$P(X = x_k)$ : 사건의 확률 (Probability of an Event)**
        * 주인공: 확률 함수(확률 측도) **$P$**
        * 의미: 확률변수 $X$가 실수 $x_k$를 가질 **'사건(집합)'**이 일어날 확률임.
        * 관점: 표본 공간의 부분집합을 다루는 **원천적인 확률론/집합론의 관점**임.
    2. **$P_X(x_k)$ : PMF 함수의 함숫값 (Value of the PMF)**
        * 주인공: 확률변수 $X$의 확률질량함수 **$P_X$**
        * 의미: 함수 $P_X$에 실수값 $x_k$를 대입했을 때 나타나는 **'함숫값(높이)'**임.
        * 관점: 복잡한 집합 구조를 배제하고 오직 실수 차원에서만 다루는 **해석학/대수학적 관점**임.
* **수학적 약속 (정의)**:
    * 두 표현은 결국 동일한 값을 가지도록 다음과 같이 정의(동치)되어 사용됨.
        $$P_X(x_k) \triangleq P(X = x_k)$$

## 6. Probability(확률), Mean(평균/기댓값), Average(표본평균)는 무엇이 다른가?
* **질문의 요지**: 일상에서 혼용되는 세 용어는 통계학적으로 어떻게 구분되는가?
* **핵심 요약 및 차이**:
    1. **Probability (확률)**
        * 개념: 특정 **사건(Event)**이 발생할 **'가능성'**을 의미함.
        * 특징: 항상 $0$과 $1$ 사이의 실수값을 가짐.
        * 예시: 주사위를 한 번 던졌을 때 6이 나올 확률은 $\frac{1}{6}$임.
    2. **Mean (평균 / 기댓값, Expected Value)**
        * 개념: 확률변수 또는 모집단 전체의 **이론적인 평균(무게중심)**임.
        * 특징: 확률분포의 성질을 이용해 데이터 관측 없이 계산할 수 있는 고정된 상수(모수)임.
        * 예시: 공정한 주사위의 이론적인 평균(기댓값)은 $3.5$임.
    3. **Average (평균 / 표본평균, Sample Mean)**
        * 개념: 실제로 관측 및 수집한 **'표본 데이터들의 산술 평균'**임.
        * 특징: 매 시행(실험)마다 수집되는 데이터에 따라 값이 변하는 가변적인 수치(통계량)임.
        * 예시: 주사위를 5번 던져 $\{1, 3, 6, 2, 4\}$를 얻었을 때 이 표본들의 평균은 $3.2$임.
* **결론**:
    * **Expected Value(기댓값) = Mean(이론적 평균)**은 수학적 동의어임.
    * 학술적인 맥락에서 **Mean(이론의 세계)**과 **Average(현실 표본의 세계)**는 엄격하게 분리하여 사용해야 함.
