## 📁 Recommended Structure
We recommend creating a separate note file for each chapter to keep things organized:
- `chapter01_basic_concepts.md` (Template already created)
- `chapter02_combinatorics.md`
- `chapter03_discrete_rvs.md`
- ...

## ✍️ Note Taking Template
You can copy the structure inside `chapter01_basic_concepts.md` to create note files for subsequent chapters.
Include:
1. **Core Concepts:** Key definitions and formulas.
2. **Important Proofs:** Explanations of how key theorems are derived.
3. **Problem Log:** Notes on tough problems you solved.
4. **Key Takeaways:** A summary in your own words.
